# Portfolio Evidence Guide

This repository is an original simulated enterprise lab. It does not claim access to a real employer production environment. Every artifact is designed to demonstrate the same reasoning, controls, documentation, and automation expected in entry-level cloud operations, infrastructure, systems administration, and IT support work.

## Interview demonstration path
1. Explain the hybrid architecture and identity flow.
2. Walk through one P2 ticket from alert to validation.
3. Demonstrate one PowerShell and one Bash health check.
4. Explain change approval, rollback, and evidence requirements.
5. Review KPI trends and propose one problem-management action.
