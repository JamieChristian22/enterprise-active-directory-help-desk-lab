# Security Hardening Baseline

- MFA for remote and privileged access
- Separate administrative accounts
- Endpoint encryption and secure boot
- Host firewall enabled
- Least-privilege role groups
- Disable legacy authentication and unused protocols
- Centralized logging and time synchronization
- Critical patch SLA of 14 days
- Quarterly privileged-access review
- Managed secrets; no credentials in scripts or repositories
- Tested backups protected from routine administrator deletion
