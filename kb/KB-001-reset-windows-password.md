# KB-001: Change Your Windows Password

1. Connect to the company network or VPN.
2. Press `Ctrl+Alt+Delete` and select **Change a password**.
3. Enter your old password and the new password twice.
4. Lock and unlock the computer using the new password.
5. Update the saved password in VPN and mobile mail apps.

Contact the service desk if the account is locked or the old password is unknown. Never send passwords through email or chat.
