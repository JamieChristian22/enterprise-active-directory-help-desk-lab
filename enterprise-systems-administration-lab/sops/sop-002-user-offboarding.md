# SOP 002: User offboarding

## Objective
Standardize user offboarding with approvals, validation, and audit evidence.

## Roles
Requester, approver, systems administrator, service owner.

## Procedure
1. Verify approved request and required details.
2. Capture pre-change state.
3. Execute using least privilege.
4. Validate the outcome.
5. Update CMDB, ticket, and documentation.

## Control Evidence
Ticket ID, approvals, timestamps, commands, validation result, and reviewer.
