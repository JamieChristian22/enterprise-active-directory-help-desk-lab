# KB-036: Monitoring Agent Failure

## Purpose
Provide a repeatable support and administration procedure for monitoring agent failure.

## Procedure
1. Confirm scope and impact.
2. Review recent changes, logs, health, dependencies, and permissions.
3. Apply the least disruptive approved remediation.
4. Validate with monitoring and an end-user or service test.
5. Document root cause, evidence, and prevention.

## Escalation
Escalate for security impact, repeated failure, data loss risk, or change outside approved authority.
