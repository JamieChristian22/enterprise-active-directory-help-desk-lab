- Tiered Active Directory administration model and privileged access controls
- GPO design, troubleshooting, and rollback evidence
- AD CS / PKI certificate lifecycle operations
- WSUS-style patch rings and compliance reporting
- Hyper-V cluster and VMware operations documentation
- Microsoft Entra Connect hybrid identity troubleshooting
- Configuration management with Ansible and PowerShell DSC
- Windows Event Forwarding, Sysmon, Prometheus, and Grafana monitoring
- Disaster recovery exercises with measured RTO/RPO
- Capacity planning, vulnerability remediation, and service-level reporting
