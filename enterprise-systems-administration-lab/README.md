# Enterprise Systems Administration Lab

A production-style, simulated enterprise systems administration portfolio covering Windows Server, Active Directory, Linux, virtualization, Microsoft 365, identity, networking, monitoring, backup, patching, security, incident response, change management, and automation.

> All organizations, users, systems, tickets, and datasets are synthetic and created for portfolio demonstration.

## Target Roles
- Systems Administrator I
- Junior Systems Administrator
- Infrastructure Engineer I
- IT Operations Engineer
- IT Support Engineer I
- Technical Support Engineer I
- Cloud Operations Engineer

## Environment
- 2 Active Directory domain controllers
- Windows Server file, print, application, WSUS, and management servers
- Ubuntu and RHEL application and utility servers
- Hyper-V and VMware-style virtualization operations
- Microsoft 365 and Entra ID administration scenarios
- Central monitoring, backups, patching, and vulnerability remediation

## Portfolio Evidence
- 120 resolved systems administration tickets
- 40 knowledge-base articles
- 30 runbooks
- 20 SOPs
- PowerShell, Bash, and Python automation
- 15 realistic operational datasets
- Incident reports, RCAs, change records, security baselines, and executive reports
- Resume bullets, STAR stories, and technical interview questions
