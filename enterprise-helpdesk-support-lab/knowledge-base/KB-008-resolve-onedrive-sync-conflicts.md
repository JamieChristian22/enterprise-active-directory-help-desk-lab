# KB-008: Resolve OneDrive sync conflicts

## Purpose
Provide a repeatable, secure support procedure for resolve onedrive sync conflicts.

## Prerequisites
Confirm user identity, business impact, device ownership, and authorization. Preserve evidence before making changes.

## Procedure
1. Confirm the exact symptom and timestamp.
2. Check service health and determine whether the issue is isolated or widespread.
3. Collect relevant configuration, logs, and error messages.
4. Apply the lowest-risk supported remediation.
5. Test the original user workflow.
6. Record actions, outcome, and any follow-up.

## Escalate when
The issue affects multiple users, requires privileged access beyond the service desk role, involves suspected compromise, or returns after standard remediation.

## Validation
The user confirms the original task works and the technician records objective evidence.
