# Enterprise Help Desk & IT Support Lab

A job-ready portfolio project demonstrating Tier 1/Tier 2 service desk, desktop support, Microsoft 365, Entra ID, Intune, Windows, networking, endpoint security, ITIL operations, customer communication, and automation.

## Business scenario
Northstar Retail Group is a 1,250-user hybrid company with three offices and remote employees. This repository simulates the daily workload of an IT Support Engineer I, Service Desk Analyst, Technical Support Engineer I, Help Desk Technician I, IT Support Associate, and Junior Systems Administrator.

## Evidence included
- 120 fully resolved support tickets with troubleshooting, customer updates, root cause, resolution, and prevention
- 45 knowledge-base articles
- 30 operational runbooks
- 20 SOPs
- 18 PowerShell automations, 8 Bash utilities, and 5 Python analytics tools
- 10 complete enterprise datasets covering tickets, endpoints, users, assets, software, SLA, changes, and incidents
- Incident, problem, change, onboarding, offboarding, escalation, and security workflows
- Resume bullets, STAR stories, interview questions, skills matrix, and portfolio presentation guide

## Core technologies
Windows 11, Windows Server, Active Directory, Entra ID, Microsoft 365, Exchange Online, Teams, OneDrive, SharePoint, Intune, Autopilot, BitLocker, Defender, PowerShell, Bash, Python, DNS, DHCP, TCP/IP, VPN, RDP, ServiceNow-style ITSM, Jira Service Management concepts, GitHub Actions.

## Quick start
1. Review `docs/environment-overview.md` and `diagrams/support-architecture.md`.
2. Open `tickets/ticket-register.csv` and any ticket file under `tickets/cases/`.
3. Run `python scripts/python/analyze_tickets.py` to generate support KPIs.
4. Use the runbooks, KB articles, and SOPs during interview walkthroughs.

## Portfolio positioning
This is a synthetic lab, not production customer data. Every record is original and designed to demonstrate repeatable troubleshooting, documentation quality, customer communication, security awareness, SLA management, and automation.
