# Environment Overview

Northstar Retail Group supports 1,250 identities, 980 Windows endpoints, 120 mobile devices, 62 servers, Microsoft 365 E3, Entra ID P1, Intune, Defender for Endpoint, Cisco AnyConnect VPN, and a segmented corporate network.

Support hours are 06:00–22:00 Eastern, Monday through Friday, with on-call coverage for Priority 1 incidents. Tier 1 owns first response, identity verification, common remediation, documentation, and customer communication. Tier 2 owns complex endpoint, Microsoft 365, networking, and application issues. Infrastructure and Security receive escalations based on documented criteria.
