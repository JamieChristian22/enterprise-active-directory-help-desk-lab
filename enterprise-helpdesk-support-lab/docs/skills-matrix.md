# Role-to-Evidence Skills Matrix

| Role | Evidence |
|---|---|
| Help Desk Technician I | ticket cases, KB articles, password/MFA/VPN/printer troubleshooting |
| Service Desk Analyst | SLA dashboard, categorization, incident and request workflows |
| IT Support Associate | onboarding, endpoint setup, software installation, customer communication |
| Technical Support Engineer I | log analysis, networking, Microsoft 365, root-cause documentation |
| Support Engineer I | automation, escalation packages, problem management, metrics |
| Junior Systems Administrator | AD, Group Policy, Microsoft 365, Intune, patching, access reviews |
