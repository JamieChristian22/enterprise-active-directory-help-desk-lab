# SOP-09: Secret rotation

## Purpose
Standardize secret rotation for the Northstar Retail cloud environment.

## Procedure
1. Confirm authorization and maintenance window.
2. Record pre-change state and dependencies.
3. Execute documented steps using least privilege.
4. Validate technical and business acceptance criteria.
5. Update the ticket, CMDB, and evidence repository.
6. Escalate exceptions and document lessons learned.

## Control
No production modification is performed without an approved change or documented emergency authorization.
