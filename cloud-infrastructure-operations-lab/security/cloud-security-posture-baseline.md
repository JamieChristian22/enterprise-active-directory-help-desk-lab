# Cloud Security Posture Baseline
- Centralized audit logging with protected retention.
- MFA and phishing-resistant authentication for privileged access.
- No standing administrator access; use time-bound elevation.
- Public storage and unrestricted administrative ports prohibited.
- Encryption required in transit and at rest.
- Critical findings remediated within 7 days; high within 30 days.
- Container images scanned before deployment; privileged containers denied.
- Quarterly access reviews and monthly key/secret age reviews.
