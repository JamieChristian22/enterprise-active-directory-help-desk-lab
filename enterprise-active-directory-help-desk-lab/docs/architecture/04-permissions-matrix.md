# Permissions Matrix

| Resource | Finance | HR | Operations | IT Help Desk | IT Admins |
|---|---|---|---|---|---|
| Finance share | Modify | None | None | Read for support | Full Control |
| HR share | None | Modify | None | No standing access | Full Control |
| Operations share | None | None | Modify | Read for support | Full Control |
| Public share | Modify | Modify | Modify | Modify | Full Control |
| Printer management | Print | Print | Print | Manage documents | Manage printers |
| AD password reset | None | None | None | Delegated on standard users | Full |
| Group membership changes | None | None | None | Approved standard groups | Full |

## Controls

- Access requires a ticket and manager approval.
- HR share access requires HR director approval.
- Help desk staff do not receive Domain Admin membership.
- Temporary access includes an expiration date and review owner.
- Access removals are completed immediately during offboarding.
