# KB-003: Reconnect a Shared Drive

1. Connect to the office network or VPN.
2. Open File Explorer and select **This PC**.
3. Choose **Map network drive**.
4. Enter the approved path, such as `\NS-FS01\Public`.
5. Select **Reconnect at sign-in** and finish.

An access-denied message requires an approved access request; repeated attempts will not grant permission.
