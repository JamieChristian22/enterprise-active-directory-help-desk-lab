# KB-002: Connect to Northstar VPN

1. Confirm internet access.
2. Open **Northstar Secure VPN** from the Start menu.
3. Select `vpn.northstar.example`.
4. Sign in with your Northstar username and current password.
5. Approve the MFA prompt you initiated.
6. Confirm the status reads **Connected** before opening shared drives.

Report unexpected MFA prompts immediately.
