# Runbook: Printer Not Printing

1. Confirm whether one user, one printer, or all printers are affected.
2. Verify power, display errors, paper, toner, and network cable.
3. Ping the printer and open its web interface if authorized.
4. Clear the local queue and restart the Print Spooler.
5. Test the server queue on `NS-PRN01`.
6. Remove and reconnect the shared printer.
7. Validate the approved driver and default paper settings.
8. Print a Windows test page.
9. Escalate hardware faults, recurring spooler crashes, or server-wide queue failures.
