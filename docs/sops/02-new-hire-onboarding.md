# SOP: New Hire Onboarding

## Required input
Approved ticket, legal name, start date, job title, department, manager, work location, license needs, and access approvals.

## Procedure
1. Validate the request against HR’s onboarding record.
2. Create the AD account in the correct department OU.
3. Assign a unique username and primary email naming convention.
4. Add department and approved resource groups.
5. Create the home folder and apply user-only Modify permission.
6. Assign the standard Microsoft 365 license through the approved admin workflow.
7. Prepare and domain-join the assigned device.
8. Apply updates, BitLocker, endpoint protection, approved software, and asset tag.
9. Test login, DNS resolution, mapped drives, printing, VPN, and Microsoft 365.
10. Send the manager a completion summary and first-day instructions.
11. Record asset serial number, assigned user, date, and technician.
