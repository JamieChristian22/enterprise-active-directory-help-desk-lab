# SOP: File Share Access Request

1. Confirm resource name and requested access level.
2. Verify manager approval and data-owner approval.
3. Check for a matching authorized security group.
4. Add the user to the group; do not assign direct NTFS permissions.
5. Ask the user to sign out and back in or refresh their Kerberos ticket.
6. Confirm access without exposing unauthorized data.
7. Record group name, approver, technician, and completion time.
8. Set an expiration date for temporary access.
