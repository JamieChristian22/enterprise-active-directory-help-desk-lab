# Service Desk Metrics Definition

- **SLA attainment:** Resolved tickets within the documented priority target divided by all resolved tickets.
- **Average resolution time:** Mean elapsed minutes from opened to resolved.
- **First-contact resolution:** Tickets solved without reassignment or escalation divided by resolved tickets.
- **Backlog:** Open tickets not in a resolved or closed state.
- **Priority distribution:** Ticket count grouped by P1–P4.
- **Category volume:** Ticket count grouped by support category.

The included dataset is synthetic and is intended only for portfolio analysis and dashboard practice.
